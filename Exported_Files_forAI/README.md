# Exported Files for AI

Pacchetto canonico generato da BlueprintJSONExporter per analisi AI.

## Contenuto

- `GLOBAL_INDEX.json`: indice strutturato globale.
- `GLOBAL_SEARCH.jsonl`: indice di ricerca line-delimited.
- `GLOBAL_SYMBOL_INDEX.jsonl`: simboli globali.
- `PATH_MAP.json`: mappa da path corti a nomi e path Unreal originali.
- `Assets/Blueprints`, `Assets/DataTables`, `Assets/Structs`, `Assets/Enums`: export per asset.
- `EXPORT_MANIFEST.json`: metadati e stato di completezza.
